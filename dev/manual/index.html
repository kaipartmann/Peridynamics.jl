Manual

Typical workflow

Peridynamics.jl simulations follow this short workflow:

  1. Create a job
  2. Submit the job

How to create a job

Currently two different simulation job types are available:

  • PDSingleBodyAnalysis
    • Define the spatial discretization - create a PointCloud
    • Define the material, see BondBasedMaterial
    • Define the temporal discretization with TimeDiscretization
    • Define export settings with ExportSettings
    • Add predefined cracks with PreCrack
    • Add boundary conditions, e.g. VelocityBC, ForceDensityBC, PosDepVelBC
    • Add initial conditions, e.g. VelocityIC
  • PDContactAnalysis
    • Create a BodySetup for every body (similar to PDSingleBodyAnalysis)
    • Define the Contact between the different bodies
    • Define the temporal discretization with TimeDiscretization
    • Define export settings with ExportSettings
