Contact analysis

Short guide on how to run a peridynamic analysis with contact of multiple bodies.

Steps

1. Define the bodies

To define the bodies, create BodySetup instances. Each BodySetup is created with step 1-4 from single body analysis guide. Optionally choose if a body should contribute to the automatic calculation of the stable time step with the calc_timestep keyword. Be careful if you use this option!

2. Define contact

Define the contact of two bodies by creating a Contact instance.

3. Choose temporal discretization settings

Create a VelocityVerlet instance.

4. Choose export settings

Create a ExportSettings instance.

5. Create the simulation job

Create a PDContactAnalysis instance and choose a simulation name.

6. Run the simulation job

Run the simulation with the submit function.
