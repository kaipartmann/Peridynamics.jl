Single body analysis

Short guide on how to run a peridynamics analysis with only one body.

Steps

1. Define a point cloud

Please refer to the guide on how to use point clouds.

2. Set material properties

Please refer to the guide on how to use material formulations.

3. Create predefined cracks [optional]

Please refer to the guide on how to use predefined cracks.

4. Assign boundary and initial conditions [optional]

Boundary conditions, updated every time step
  • Velocity boundary condition, VelocityBC
  • Force density boundary condition ForceDensityBC
  • Position dependend velocity boundary condition, PosDepVelBC
Initial conditions, only assigned before time-loop
  • Velocity initial condition VelocityIC

5. Assign a temporal discretization

Create either a VelocityVerlet or a DynamicRelaxation instance.

6. Choose export settings

Create a ExportSettings instance.

7. Create the simulation job

Create a PDSingleBodyAnalysis instance and choose a simulation name.

8. Run the simulation job

Run the simulation with the submit function.
